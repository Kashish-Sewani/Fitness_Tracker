# 🏋️ AI Fitness Tracker App

A full-stack AI-powered fitness application that helps users set fitness goals, track calories, log workouts, and analyze food images using Google Gemini AI.

## 🎯 Features

✅ **User Authentication** - Secure sign up and sign in with JWT tokens  
✅ **Profile Management** - Set and update user fitness profile (age, weight, height, goals)  
✅ **Daily Fitness Goals** - Set calorie intake and burn targets  
✅ **Food Tracking** - Log daily food intake with calorie counts  
✅ **Activity Logging** - Track workouts and calories burned  
✅ **AI Food Analysis** - Upload food images to analyze with Google Gemini AI  
✅ **Smart Calorie Calculator** - Automatic calorie calculation from meal type and time of day  
✅ **Dashboard Analytics** - Visual tracking of daily progress with charts  
✅ **Dark Mode Support** - Built-in light/dark theme toggle  
✅ **Responsive Design** - Fully mobile-friendly interface  
✅ **Real-time Updates** - Instant stats and progress updates

## 🚀 Getting Started

### Installation

#### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/fitness-tracker.git
cd Fitness_Tracker
```

#### 2. Setup Frontend

```bash
cd client
npm install
```

Create a `.env` file in the `client` directory:

```env
VITE_API_URL=http://localhost:1337
```

#### 3. Setup Backend

```bash
cd server
npm install
```

## Environment Variables

Create a `.env` file inside the `server` folder:

```env
HOST=0.0.0.0
PORT=1337

APP_KEYS=your_app_keys
API_TOKEN_SALT=your_api_token_salt
ADMIN_JWT_SECRET=your_admin_jwt_secret
TRANSFER_TOKEN_SALT=your_transfer_token_salt
ENCRYPTION_KEY=your_encryption_key
JWT_SECRET=your_jwt_secret

DATABASE_CLIENT=sqlite
DATABASE_FILENAME=.tmp/data.db

GEMINI_API_KEY=your_google_gemini_api_key
```

Most of these variables are automatically generated by Strapi during setup. You can create a new Strapi instance or configure your own values if running the project locally.

## 💻 Running the Application

### Development Mode

**Terminal 1 - Start Backend:**

```bash
cd server
npm run develop
```

Backend will be available at `http://localhost:1337`

**Terminal 2 - Start Frontend:**

```bash
cd client
npm run dev
```

Frontend will be available at `http://localhost:5173`


## 🤖 AI Features

### Google Gemini Food Analysis

The app uses Google Gemini AI to analyze food images:

1. Upload a food photo
2. Gemini AI identifies the food item
3. Automatic calorie calculation
4. Entry is added to your food log
5. Dashboard updates in real-time

**Note:** Meal type is automatically determined based on time of day:
- 00:00 - 11:59: Breakfast
- 12:00 - 15:59: Lunch
- 16:00 - 17:59: Snack
- 18:00 - 23:59: Dinner
